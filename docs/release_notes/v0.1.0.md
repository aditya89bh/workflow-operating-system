# Release Notes — v0.1.0

The first public release of the **Workflow Operating System**: a lightweight,
deterministic, dependency-free Python library for modeling, running, and learning
from workflows.

## Highlights

- **End-to-end pipeline** across nine layers: workflow engine → memory →
  decisions → SOPs → approvals → exceptions → multi-agent collaboration →
  analytics → organizational learning.
- **Ten showcase demos** runnable from the CLI, covering realistic domains
  (employee onboarding, procurement, incident management, customer onboarding)
  and cross-layer integrations.
- **One-command tour:** `workflow-os demo-all`.
- **Documentation:** getting started, a guided tutorial, per-layer guides, and
  architecture diagrams.
- **Benchmark suite** for workflow execution, memory retrieval, and analytics
  generation.
- **Deterministic by design:** no machine learning, prediction, or external
  services anywhere in the stack.

## Quality

- ~596 tests passing
- `ruff` lint clean, `mypy` type-clean (`src`)
- Builds an sdist and wheel with `python -m build`

## Install

```bash
pip install -e .
workflow-os demo-all
```

## Try it

```bash
workflow-os demo employee-onboarding
workflow-os demo organizational-learning
```
